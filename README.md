# revisit-sca-gcm


This repository contains the power-trace dataset and source code for the paper **"Revisiting Side-Channel Resistance of GCM Implementations on AVR Microcontrollers"**.

## 📌 Project Overview
This study re-evaluates the side-channel resistance of Galois/Counter Mode (GCM) implementations on low-power 8-bit AVR microcontrollers. We provide the complete framework to reproduce our experimental results, including:

* **Power-Trace Dataset:** Side-channel leakage traces collected during GCM operations.
* **Analysis Scripts:** Power Analysis and leakage assessment scripts.
